- **DISCLAIMER:** This project is a modified version of https://github.com/cowster36/mazegame.
Additional features include a scoreboard, a combat system involving swords and health points, and an AVL tree for storing player statistics.
-
- The game consists of a randomly generated maze, wherein the player (represented by a yellow/green square) competes against the computer (represented by a red square) to collect a higher number of coins. The two also engage in combat, and the game culminates when either character is eliminated. At this point, the character with the higher number of coins is declared to be the winner.
